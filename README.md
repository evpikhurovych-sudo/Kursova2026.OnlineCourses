# Kursova2026 – Платформа онлайн-курсів

## Опис
Курсовий проєкт з дисципліни **Об’єктно-орієнтоване програмування**.  
Тема: **Розроблення програмного забезпечення з використанням об’єктно-орієнтованої парадигми**.  
Предметна область: **Платформа онлайн-курсів (варіант 8)**.

Проєкт реалізовано з дотриманням принципів **SOLID**, відокремленням предметної області від інтерфейсу користувача, використанням **делегатів і подій**, а також з підтримкою збереження/завантаження даних у форматі **JSON**.

---

## Технології
- C# / .NET (ASP.NET Core MVC)
- xUnit (Unit-тести)
- JSON (збереження/завантаження колекцій даних)
- GitHub (issues, milestones)

---

## Структура рішення
- **Онлайн-курси.Веб** – Web-інтерфейс (ASP.NET Core MVC)
- **Онлайн-курси.Домен** – класи предметної області (Entities, Interfaces, Events, Exceptions)
- **Онлайн-курси.Тести** – unit-тести для класів предметної області
- **Курсова2026.Онлайн-курси.sln** – файл рішення

> Предметна область повністю відокремлена від UI.

---

## Основні можливості (план)
- Ролі користувачів: **Адміністратор / Викладач / Студент**
- Керування курсами та уроками
- Запис на курс (enrollment) та відстеження прогресу
- Події предметної області (наприклад, зміна прогресу)
- Експорт/імпорт даних у **.json**

---

## Як запустити
1. Відкрити solution `Курсова2026.Онлайн-курси.sln` у Visual Studio
2. Встановити стартовим проєкт **Онлайн-курси.Веб**
3. Запустити (Run)
 запускати по HTTP.
